# PWA Icons

The source artwork for the Hossein Starship PWA icon lives at:

`public/pwa-icon.svg`

Generated assets:

- `public/apple-touch-icon.png`
- `public/icon-192.png`
- `public/icon-512.png`
- `public/favicon-32x32.png`
- `public/favicon-16x16.png`
- `public/maskable-icon-512.png`

Regenerate the icon set with:

```bash
npm run icons:pwa
```

Verify the icon files with:

```bash
npm run icons:verify
```
